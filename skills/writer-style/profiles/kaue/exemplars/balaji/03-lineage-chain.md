A dollar that lives on a network is an old idea, and it reads best as a chain of near-misses, each missing one ingredient the next supplied. Early digital cash moved value but leaned on a company's server to stop double-spends — settlement with no neutral party to settle on. Reserve-backed notes fixed the trust in the token by parking dollars in a bank, then moved the trust to whoever held that account — a claim no one could check. Putting collateral on-chain made the backing checkable by anyone, but tied the coin's value to volatile assets — transparent, not stable. Each attempt named the gap for the next: a neutral ledger, then a verifiable reserve, then a buffer against the collateral swinging. None of it is settled; the missing piece is just easier to name than to supply.
